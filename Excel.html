    • O que é?

    o Excel é um software voltado para a criação de planilhas eletrônicas. Ele foi desenvolvido como um software especialmente para empresas e que proporciona desde controle de estoques até relatórios financeiros. Ele faz parte do Pacote Office, que oferece produtos para criar textos (Microsoft Word), apresentações (Microsoft Powerpoint), entre outras funcionalidades

    O Microsoft Excel é uma excelente ferramenta para complementar o currículo, e pode ser automatizado de diferentes formas, mas nem todas as pessoas têm domínio sobre ele como têm sobre o Word ou Powerpoint.

    Funções

    é possível utilizá-las para fazer cálculos, analisar dados e cruzar diferentes planilhas; Lembrando que apesar do Excel possuir diversas funcionalidades, ele não possui tudo o que um empreendedor precisa para gerir seu negócio de forma prática e eficaz.

    Principais Atalhos:

    Atalho Função
    CTRL + A abre um arquivo já existente
    CTRL + B salva o documento
    CTRL + C copia o texto selecionado
    CTRL + V cola o último texto copiado
    CTRL + P imprime o documento
    CTRL + Z desfaz a última ação
    CTRL + Y refaz a última ação
    CTRL + S sublinha o texto
    CTRL + I formata o texto selecionado para ítalico
    CTRL + N formata o texto selecionado para negrito
    CTRL + ALT + V cola o texto copiado sem formatação
