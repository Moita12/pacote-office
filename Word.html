    • Word

    O Microsoft Word é um programa de edição de texto extremamente popular por vir normalmente instalado em todos os computadores com sistema operacional Windows. Já teve algumas versões distintas para ampliar suas funcionalidades e também aceita softwares complementares para ampliar suas ferramentas. No geral, serve para criação de textos corridos, com algumas opções de formatação e edição, inserção de imagens e tabelas. Quem precisa fazer um contrato, monografia ou quer escrever um livro irá usar o Microsoft Word.

    No mundo corporativo, é mais usado para produção de documentos legais.

    Principais atalhos do Word:

    Atalho Função
    CTRL + A abre um arquivo já existente
    CTRL + B salva o documento
    CTRL + C copia o texto selecionado
    CTRL + V cola o último texto copiado
    CTRL + P imprime o documento
    CTRL + Z desfaz a última ação
    CTRL + Y refaz a última ação
    CTRL + S sublinha o texto
    CTRL + I formata o texto selecionado para ítalico
    CTRL + N formata o texto selecionado para negrito
    CTRL + ALT + V cola o texto copiado sem formatação
