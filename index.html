    • O que é o pacote Office?

    O Pacote Microsoft Office é um compilado de aplicativos para escritório que se tornou indispensável para os seus negócios. Nele estão incluídas as principais ferramentas, como o Word, o PowerPoint e o Excel. No entanto, é possível adquirir mais aplicativos a depender da versão paga, como o Teams, Exchange e Access.

    Para ter acesso a todos os benefícios do pacote Office, é necessário pagar planos anuais ou mensais de uso – que incluem benefícios como backup na nuvem para todos os documentos, planos empresariais para diferentes computadores e algumas usabilidades a mais.
